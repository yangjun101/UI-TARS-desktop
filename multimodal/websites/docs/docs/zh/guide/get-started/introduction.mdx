# Agent TARS

## What's Agent TARS?

Agent TARS 是一个开源的 Multimodal AI Agent，旨在通过丰富的多模态能力（如 [GUI Agent](../basic/browser.mdx#visual-grounding)、[Vision](../basic/vision.md)） ，并与各种现实世界工具无缝集成，其内置了常用的工具（[Search](../basic//search.md)，[Browser](../basic/browser.md)、[File](../basic/file.md)、[Command](../basic/command.md) 等），来不断探索一种能够更接近人类完成任务的工作形态。

---

Agent TARS 同时提供 CLI 和 SDK。CLI 非常适合快速体验 Agent TARS 提供的功能，而 SDK 则旨在帮助您使用 Agent TARS SDK 构建自己的 Agent。请根据您的具体用例进行选择：
